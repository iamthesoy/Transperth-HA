# Brand assets

This integration ships with `icon.png` for Home Assistant (local `brand/` folder).

To use the official Transperth bus GIF instead:

1. Open [bus.gif](https://136213.mobi/App_Themes/TP_Mobi/icons/bus.gif) in your browser and save the file.
2. Convert it to a square PNG (256×256 recommended), or keep GIF if your Home Assistant version accepts it for custom integrations.
3. Replace `icon.png` in this folder (or add `logo.png` alongside if you use both).

If direct download returns 403, use a normal browser session (the site may block scripted downloads).
